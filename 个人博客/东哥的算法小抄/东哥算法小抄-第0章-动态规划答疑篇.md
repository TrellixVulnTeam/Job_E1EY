# 最优子结构
可以从⼦问题的最优结果推出更⼤规模问题的最优结果，最优⼦结构并不是动态规划独有的⼀种性质，能求最值的问题⼤部分都具有这个性质；但反过来，最优⼦结构
性质作为动态规划问题的**必要条件**，⼀定是让你求最值的。

# DP数组的遍历方向

## 正向遍历

```
for row in [0:rows-1]:
    for col in [0:cols-1]:
        pass
```

## 反向遍历

```
for row in [rows-1:0:-1]:
    for col in [cols-1:0:-1]:
        pass
```

## 斜向遍历

```
for (int l = 2; l <= n; l++) 
{
    for (int i = 0; i <= n - l; i++) 
    {
        int j = l + i - 1;
        // 计算 dp[i][j]
    }
}
```

* 遍历的过程中，所需的状态必须是已经计算出来的。
* 遍历的终点必须是存储结果的那个位置。
